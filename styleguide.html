Pre-formatted text

The pre element represents a block of pre-formatted text, in which structure is represented by typographic conventions rather than by elements. Code inside of the code element is often found inside a pre tag in order to preserve the syntax structure of the coding language.

Text-level Semantics

There are a number of inline HTML elements you may use anywhere within other elements.

Links and anchors

The a element is used to link to other pages on your website or to external pages on the web.

This is a link to Google

Stressed emphasis

The em element is used to denote text with stressed emphasis.

You really need to watch Downton Abbey.

Stong Importance

The strong element is used to denote text with strong importance. Where bolding is used for stylistic differentiation, the b element may be preferable. Example:

Do not go in there!

Tabular Data

Tables should be used when displaying tabular data. The thead, tfoot and tbody elements enable you to group rows within each a table.

If you use these elements, you must use every element. They should appear in this order: thead, tfoot and tbody, so that browsers can render the foot before receiving all the data. You must use these tags within the table element.
